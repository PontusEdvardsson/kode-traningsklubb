# Kode Träningsklubb

En statisk one-page-sida för Kode Träningsklubb.

## Filer

- `index.html` innehåller hela sidans innehåll.
- `styles.css` innehåller layout, färger och responsiv design.
- `assets/kode-traningsklubb.jpg` är logotypen.

## Publicera på GitHub Pages

1. Skapa ett repo som heter `kode-traningsklubb`.
2. Lägg in filerna i repots rot.
3. Pusha till `main`.
4. Aktivera GitHub Pages i repo-inställningarna med källa `Deploy from a branch`.
5. Välj branch `main` och mapp `/ (root)`.

Sidan innehåller uppgifter sammanställda från föreningens informationsinlägg, Messenger-svar och medlemsintyget.
